# Testigos 🚗

Esta es una aplicación de Streamlit diseñada para gestionar y descargar testigos de forma automatica por marca competencia.

## 🛠️ Instalación y Uso Local
1. Clona el repositorio.
2. Instala las dependencias: `pip install streamlit` (y las que use tu código).
3. Ejecuta la app: `streamlit run app_testigos.py`

## 🚀 Despliegue
La app está desplegada en Streamlit Community Cloud.
